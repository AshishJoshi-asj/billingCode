Step of generating the billing reports generated csv billing file :
1- create a folder with the specified name. 
2- put the csv file with name "awsBilling.csv" into this folder.
3- change the "max_execution_time=900" in php.ini file of apache
4- run the billingAmazon.php file in apache 
5- Output will be generated with the file name "billingReport.xls"
